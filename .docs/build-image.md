# Usage

## build-image

```text
$ build-image -h
Usage:
    build-image [options]
Options:
    --debug               - run in debug mode
    -h, --help            - show help message and exit
    --update              - update scripts
    -V, --version         - show version

    --force               - force build
    --push                - push image
```

> Note: options order is important \
> [Notes](.docs/update_option.md) on using `--update` option
